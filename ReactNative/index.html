Comece Aprenda o Básico

Crie aplicativos nativos para Android, iOS
e outros usando o React

O React Native traz o melhor do desenvolvimento com o React para o desenvolvimento nativo.

É uma biblioteca JavaScript de primeira classe para construir interfaces de usuário.

Escrito em JavaScript, renderizado com código nativo. Primitivas do React renderizam na interface nativa da plataforma, ou seja, seu app usa as mesmas APIs nativas da plataforma que outros apps usam.

Desenvolvimento nativo para todos

O React Native permite que você crie aplicativos verdadeiramente nativos e não compromete a experiência dos seus usuários. Ele fornece um conjunto central de componentes nativos independentes da plataforma, como View, Text Image e que mapeiam diretamente para os blocos nativos da interface da plataforma.

Comece com uma estrutura

O React Native traz o paradigma de programação React para plataformas como Android e iOS. Não prescreve como fazer roteamento, nem como acessar cada uma das inúmeras APIs da plataforma.

Para construir um novo app com React Native, recomendamos um framework como o Expo.

Roteamento baseado em arquivos

Crie telas de pilha, modais, gavetas e abas com o mínimo de boilerplate usando seu sistema de arquivos.

Use qualquer biblioteca, SDK ou código nativo

Gere mudanças nativas ou escreva seu próprio código nativo. Use mais de 50 módulos para criar seu app.

Ferramentas para desenvolvedores

Comece rapidamente com o Expo Go, depois continue com o expo- dev-client: um módulo que adiciona as ferramentas da Expo a aplicativos que exigem mudanças nativas.
